Underwater Image Enhancement Results Bundle
Date: 2026-05-12

This folder groups the generated project artifacts into one shareable package.

Contents
01_training
- training_log.csv: exported training log from logs/train_log.csv
- loss_curve.png: training loss visualization
- training_curves.png: training curve figure

02_evaluation
- summary_metrics: aggregate and per-image evaluation outputs for EUVP, UIEB, and RUIE
- qualitative_outputs/UIEB: enhanced UIEB result sets
- qualitative_outputs/RUIE: enhanced RUIE result sets
- qualitative_outputs/validation: validation comparison/enhanced images
- qualitative_outputs/batch: batch inference outputs
- qualitative_outputs/single: single-image comparison outputs
- qualitative_outputs/samples: curated sample outputs

03_metric_comparison
- ablation: ablation study tables and CSVs
- clahe: CLAHE baseline comparison CSVs and summary

04_figures
- publication and architecture figures

05_existing_archives
- previously exported ZIP archives copied for completeness

Source files were copied into this bundle so the original repo layout remains unchanged.
