3. METHODOLOGIES

This chapter explains the methods and deep learning concepts related to the proposed underwater image enhancement system. Methodology is important because it describes the technical foundation used to solve the problem. In deep learning, different neural network models are designed for different types of data. Some are best for images, some for sequential data, and some for time-dependent patterns. In this project, the main task is underwater image enhancement, which is an image processing problem. Therefore, Convolutional Neural Network (CNN) is the most relevant algorithm. Recurrent Neural Network (RNN) and Long Short-Term Memory (LSTM) are also studied as part of deep learning methodologies because they are important architectures in artificial intelligence, although they are more suitable for sequential and temporal data than static image enhancement.

3.1 Deep Learning Algorithms

Deep learning is a branch of machine learning that uses artificial neural networks with multiple layers to automatically learn patterns from data. Unlike traditional image processing methods, deep learning models do not rely only on manually designed rules. Instead, they learn important features directly from training examples. Deep learning is widely used in image classification, object detection, segmentation, enhancement, speech recognition, and natural language processing. In this project, deep learning is used because underwater images contain complex distortions such as colour cast, low contrast, blur, and reduced visibility, which are difficult to correct using simple rule-based techniques.

3.1.1 Convolutional Neural Network (CNN)

A Convolutional Neural Network is a deep learning model specially designed for image-related tasks. CNN is very effective because it can automatically learn spatial features such as edges, textures, patterns, colours, and object structures from images. It uses convolution layers, pooling layers, and activation functions to extract meaningful features from the input image. In image enhancement tasks, CNN learns how to transform a degraded image into a visually improved image.

For underwater image enhancement, CNN is highly suitable because it can identify local image distortions such as haze, colour imbalance, and poor contrast. It can then learn correction patterns to improve the image quality. CNN also preserves spatial information, which is very important in enhancement tasks because the position of objects and image structures must remain consistent. In this project, the enhancement model is based mainly on a CNN encoder-decoder framework, which helps extract image features, process them deeply, and reconstruct an enhanced output image. Thus, CNN is the core algorithm of the proposed system.

Main advantages of CNN:
- very effective for image processing tasks
- automatically learns image features
- preserves spatial relationships
- suitable for enhancement, restoration, and classification

3.1.2 Recurrent Neural Network (RNN)

A Recurrent Neural Network is a deep learning model mainly used for sequential data, where previous information affects the current output. Unlike CNN, RNN has a feedback mechanism that allows it to remember information from earlier steps in a sequence. This makes it useful for tasks such as language modeling, speech recognition, text generation, and time-series prediction.

RNN is not the primary algorithm for underwater image enhancement because the project deals with still images rather than sequential data. However, RNN is included in the methodology chapter to show understanding of different deep learning architectures. If the problem involved underwater video enhancement, frame-by-frame temporal learning, or sequential environmental sensor data, then RNN could become more useful. Its main limitation is that it struggles with long-term dependencies when sequences become large.

Main advantages of RNN:
- useful for sequential and time-dependent data
- can store previous information
- suitable for text, speech, and sequence modeling

3.1.3 Long Short-Term Memory (LSTM)

Long Short-Term Memory is an advanced form of RNN developed to overcome the short-memory problem of standard RNNs. LSTM uses memory cells and gating mechanisms such as input gate, forget gate, and output gate to control the flow of information. Because of this design, LSTM can learn long-range dependencies more effectively than a normal RNN.

LSTM is commonly used in speech processing, language translation, handwriting recognition, and time-series forecasting. In the context of this project, LSTM is not directly used for underwater image enhancement because the task focuses on static images. However, it is still part of deep learning methodology because it represents an important improvement over RNN and is useful when long sequential dependencies must be captured. If the project were extended to underwater video enhancement, object tracking, or temporal sequence analysis, LSTM could be considered.

Main advantages of LSTM:
- solves the vanishing memory problem of RNN
- captures long-term dependencies
- suitable for long sequential data
- more stable than basic RNN for complex sequences

Conclusion of Methodology Section

Among CNN, RNN, and LSTM, CNN is the most suitable algorithm for this project because underwater image enhancement is primarily an image-based task. RNN and LSTM are important deep learning methods, but they are more appropriate for sequential and time-series data. Therefore, the proposed system mainly relies on CNN-based learning for improving underwater image quality.
